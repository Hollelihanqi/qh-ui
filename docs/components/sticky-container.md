## API

### 插槽

| 名称    | 说明         |
| ------- | ------------ |
| default | 主要内容部分 |
| header  | 头部内容     |
| footer  | 底部内容     |

### 方法

| 名称                 | 说明                   |
| -------------------- | ---------------------- |
| getScrollbarInstance | 获取 el-scrollbar 实例 |
| scrollToBottom       | 滚动到底部             |
