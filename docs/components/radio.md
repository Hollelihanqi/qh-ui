## API

### Radio 属性

| 属性名      | 说明                          | 类型               | 默认值 |
| ----------- | ----------------------------- | ------------------ | ------ | --- |
| v-model     | 选中项的值                    | any                | []     |
| label       | IOptions 类型时对应的文本字段 | Function           | —      |
| value       | IOptions 类型时对应的值字段   | Boolean            | true   |
| options     | 数据                          | string，IOptions[] |        | —   |
| activeColor | 激活项颜色                    | string             |        | —   |
| disabled    | 是否禁用                      | Boolean            | false  | —   |
